× Music Shopping App
Technologies: Android Studio, Java, Figma, Git Control In collaboration with Clinton Philathong

Applied concepts such as proper UI design, User Input, varying view groups (i.e Recyclerview), and working with databases. × Flappy bird game on DE0 Board
Technologies: Quartus, DE0 Board In collaboration with May Sribunwongsa, Jordan Green

Designed from scratch a simple game based on flappy bird on the DE0 console using only digital logic and digital design. The game is controlled by PS/2 mouse, DIP switches, and push buttons on the DE0 board. Used programs such as Quartus, modelsim, and coded in VHDL to configure the digital design and logic design for the firmware. Incorporated game design rules such as number of lives, randomly generated pipe distances, memory storage for score, invisibility power ups, and level difficulties.
